Tentang Saya

Halo, Saya Arfan Nur Ivandi

Saya merupakan mahasiswa tingkat akhir Universitas Islam Raden Rahmat Malang dengan minat utama pada bidang teknologi. Keahlian saya meliputi pengembangan aplikasi full-stack menggunakan Express.js, Fiber Go, Flask, serta framework PHP (Laravel dan CodeIgniter), disertai kemampuan mobile development dengan Kotlin dan desktop development dengan Tkinter Python.

Selain itu, saya aktif dalam riset dan pengembangan di bidang robotik serta IoT, dengan pengalaman menggunakan ESP32, LoRa, dan Raspberry Pi Pico untuk integrasi perangkat keras dan perangkat lunak. Saya juga memiliki kompetensi dalam manajemen jaringan RT/RW Net berbasis MikroTik.

Saya berpegang pada prinsip pembelajaran berkelanjutan dan inovasi, dengan tujuan menghadirkan solusi teknologi yang bermanfaat dan berdampak positif bagi masyarakat.
